# xlog - Expense Logger

`xlog` is a CLI utility to log your expenses on to a google sheet.

## Requirements

- [uv](https://github.com/astral-sh/uv) CLI tool
- Python 3.13+

### Running (Development)

```sh
uv run xlog
```

**Note**: This would automatically install the dependencies in `pyproject.yaml` under `.venv/`.
